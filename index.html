About Me

Hi there, thank you for checking out my portfolio. I am Antony, a software developer focusing on web and cross platform mobile technologies.

I have a paticular interest in the React Native framework on mobile. On web I have worked with both frontend (React Js, Next Js, CSS, SCSS, Material UI) and backend (Python, Django, REST & GraphQl) technologies to create amazing web experiences.

A web app built on React Js and intergrated with Firebase(storage). A user uploads an image of any extension(png, jpg, jpeg..) while viewing in real-time the progress of file upload. Once the upload is complete, they get back a URL for the image.
